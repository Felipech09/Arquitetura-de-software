## Arquitetura de Software
Este repositório reúne os projetos desenvolvidos durante a aula de quinta da disciplina de "Arquitetura de Software".
Cada aula possui sua própria pasta, organizada por ordem, contendo os arquivos e códigos trabalhados em sala.

# Estrutura
- As pastas estão separadas por aula (ex.: aula1, aula2, etc.).
- Dentro de cada pasta estão os sites e exercícios correspondentes.
# Links dos projetos
- Os links para acessar os sites estão disponíveis dentro de cada pasta do projeto.
- Assim, é possível visualizar diretamente o resultado de cada atividade desenvolvida.

Observação: Este repositório é exclusivamente para fins acadêmicos, acompanhando o progresso das aulas.
